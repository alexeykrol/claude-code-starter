# Установка Claude Code Starter Framework

## Перед установкой

**Установите Node.js** (если еще не установлен):

1. Скачайте с https://nodejs.org/ (выбирайте LTS версию)
2. Установите (просто нажимайте "Далее")
3. Перезапустите терминал

**Проверьте, что Node.js установлен:**
```bash
node --version
```

---

## Установка

**1. Создайте папку проекта** (если еще не создана)

**2. Скачайте установочный файл** `init-project.sh` из корня репозитория в корень вашей папки проекта

**3. Откройте VSCode:**
   - File → Open Folder
   - Выберите папку проекта (где уже лежит `init-project.sh`)

**4. Откройте терминал в VSCode:**
   - Menu: Terminal → New Terminal

**5. Введите команду:**
```bash
./init-project.sh
```

---

## Готово!

После завершения установки прочитайте как пользоваться фреймворком в файле **FRAMEWORK_GUIDE.md**, который найдете в корне вашего проекта.
